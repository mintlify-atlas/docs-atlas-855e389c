      Quick Start

    <p className="text-base text-gray-600 dark:text-gray-400 max-w-2xl mx-auto">
      Get Connected up and running in minutes with this step-by-step guide
    </p>
  </div>

  <Steps>
    <Step title="Clone the repository">
      Start by cloning the Connected repository to your local machine:
      
      ```bash
      git clone https://github.com/nintang/cf_ai_connected.git
      cd cf_ai_connected
      ```
    </Step>
    
    <Step title="Install dependencies">
      Connected uses pnpm for package management. Install all dependencies:
      
      ```bash
      pnpm install
      pnpm build
      ```
      
      This will install dependencies for the entire monorepo and build all packages.
    </Step>
    
    <Step title="Configure API keys">
      Create a `.dev.vars` file in `apps/worker/` with your API credentials:
      
      ```env
      # Google Programmable Search Engine
      GOOGLE_API_KEY=your_google_api_key
      GOOGLE_CX=your_search_engine_id
      
      # AWS Rekognition
      AWS_ACCESS_KEY_ID=your_aws_access_key
      AWS_SECRET_ACCESS_KEY=your_aws_secret_key
      
      # OpenRouter (Gemini)
      OPENROUTER_API_KEY=your_openrouter_key
      OPENROUTER_MODEL=google/gemini-2.5-flash-lite
      ```
      
      <Note>
        You'll need API keys from Google Cloud Platform, AWS, and OpenRouter. See the [Environment Setup](/guides/environment-setup) guide for detailed instructions.
      </Note>
    </Step>
    
    <Step title="Start the development servers">
      Run the worker and web app in separate terminals:
      
      ```bash Terminal 1
      cd apps/worker
      pnpm dev
      ```
      
      ```bash Terminal 2
      cd apps/web
      pnpm dev
      ```
      
      Visit `http://localhost:3000` to see Connected in action!
    </Step>
  </Steps>
</div>
